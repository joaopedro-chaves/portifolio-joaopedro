# Portfolio Front-end

[Português](#português) | [English](#english)

---

## English
A modern, responsive, and bilingual portfolio designed to showcase web design and front-end development skills. This project was developed focusing on high-end aesthetics, accessibility, and performance.

### Inspiration
This project was inspired by [Elias Portfolio](https://elias-portfolio-222e.vercel.app/), bringing its clean and organized structure to a personalized experience.

### Technologies
- **HTML5**: Semantic and organized structure.
- **CSS3**: Layout with CSS Grid, Flexbox, and fluid typography using `clamp()`.
- **JavaScript**: UI interactivity and dynamic language switching.
- **Font Awesome**: Modern icons for social links and category badges.
- **Google Fonts**: Lexend and Roboto Mono for a premium look.

### Features
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile.
- **Internationalization (i18n)**: Seamless switching between English and Portuguese.
- **Skills Board**: Organized skill categories in a 5-column grid (Desktop).
- **Mobile Menu**: Animated hamburger menu with transitions.

### [Live demo](https://portifolio-joaopedro.vercel.app/)

### How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/joaopedro/portifolio-joaopedro.git
   ```
2. Open `index.html` in your favorite browser.
3. (Optional) Use a local server (like Live Server in VSCode) for a better experience.

### Image Optimization
Recomend for performance:

1. Conversion to WebP (Required for Performance)
The WebP format reduces file size by up to 80% without any visible loss of quality compared to PNG.

Suggested Tool: Squoosh.app (Google) or TinyPNG.

## Português
Um portfólio moderno, responsivo e bilíngue criado para demonstrar habilidades em web design e desenvolvimento front-end. Este projeto foi desenvolvido com foco em estética premium, acessibilidade e performance.

### Inspiração
Este projeto foi inspirado em [Elias Portfolio](https://elias-portfolio-222e.vercel.app/), trazendo sua estrutura limpa e organizada para uma experiência personalizada.

### Tecnologias
- **HTML5**: Estrutura semântica e organizada.
- **CSS3**: Layout com CSS Grid, Flexbox e tipografia fluida usando `clamp()`.
- **JavaScript**: Interatividade da UI e troca dinâmica de idiomas.
- **Font Awesome**: Ícones modernos para links sociais e emblemas de categoria.
- **Google Fonts**: Lexend e Roboto Mono para um visual premium.

### Funcionalidades
- **Design Responsivo**: Otimizado para Desktop, Tablet e Mobile.
- **Internacionalização (i18n)**: Troca fluida entre Inglês e Português.
- **Quadro de Habilidades**: Categorias de habilidades organizadas em uma grade de 5 colunas (Desktop).
- **Menu Mobile**: Menu hambúrguer animado com transições suaves.

### [Ver o projeto](https://portifolio-joaopedro.vercel.app/)

### Como Rodar Localmente
1. Clone este repositório:
   ```bash
   git clone https://github.com/joaopedro/portifolio-joaopedro.git
   ```
2. Abra o arquivo `index.html` no seu navegador favorito.
3. (Opcional) Use um servidor local (como a extensão Live Server do VSCode) para uma melhor experiência.

### Otimização de Imagens
Recomendo que você realize estes passos:

1. Conversão para WebP (Obrigatório para Performance)
O formato WebP reduz o tamanho do arquivo em até 80% sem perda visível de qualidade em relação ao PNG.

Ferramenta Sugerida: Squoosh.app (Google) ou TinyPNG.
